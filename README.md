# Aplicación de Lenguaje de Señas - UCV

Aplicación web para el aprendizaje de lenguaje de señas desarrollada para la Universidad César Vallejo.

## Características

- Autenticación con Google
- Galería de señas
- Juego de práctica
- Foro de documentos
- Sistema de reseñas
- Perfil de usuario

## Tecnologías

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Base de datos: MongoDB Atlas
- Hosting: GitHub Pages (frontend) y Render (backend)

## Instalación

1. Clonar el repositorio
2. Instalar dependencias: `npm install`
3. Crear archivo .env con las variables de entorno necesarias
4. Iniciar servidor: `npm start`

## Variables de Entorno

```env
MONGODB_URI=tu_uri_de_mongodb
GOOGLE_CLIENT_ID=tu_client_id_de_google
BASE_URL=https://tu-dominio.com
```

## Uso

1. Visitar la página principal
2. Iniciar sesión con Google
3. Explorar las diferentes secciones
4. Practicar con el juego interactivo
5. Compartir documentos y reseñas
